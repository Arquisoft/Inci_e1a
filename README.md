# Sistema informático de gestión de incidencias
## Módulos
### Sistema basado en microservicios, consta de los siguientes módulos
* Carga de usuarios
* Consulta y gestión de agentes
* Gestión y carga de incidencias
* Análisis y cuadro de mandos
